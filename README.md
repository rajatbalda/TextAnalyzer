# 📝 Text Analyzer

A simple web-based tool to analyze any given text for:

- Word count
- Character count (excluding spaces)
- Sentence count

## 🚀 Features

- Paste any text into the input area.
- Click **Analyze** to see real-time stats.
- Works instantly in any modern browser.

## 🛠 Built With

- HTML
- CSS
- JavaScript
